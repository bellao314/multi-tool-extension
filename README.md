# Multi-Tool Chrome Extension

## Features
- Calculator
- Timer/Stopwatch
- Notes
- Screen Recording
- Google Gemini chatbot


## Usage

Clone repository.

Run `npm install`.

Run `npm run build`.

Go to [chrome://extensions](chrome://extensions).

Turn on "Developer Mode".

Click "Load Unpackaged".

Upload `build` folder created from the `npm run build` command.

Pin the extension to the Chrome header.

Enjoy your new extension!
